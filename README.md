## github 사용법
  #### 빈 저장소 github
  1. local 에 먼저 저장소를 만들고 (git init)
  2. comit 을 한후 (commit 하기전에 .gitignore 파일을 추가)
  3. https://github.com/shinbumjun/mall2 여기를 remote 저장소로 등록
  4. 그리고 push
